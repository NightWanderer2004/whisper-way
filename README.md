## Tech Stack

### Base
- Next.js
- Tailwind CSS
- Framer Motion
- Supabase
- React Hook Form

### Advanced
- Mapbox GL — A JavaScript library for interactive, customizable vector maps
- Radix UI — A set of low-level, accessible UI components
- Sonner — A library for smooth toast notifications
- Lucide Icons — A collection of simple and consistent icons
- Workbox — A set of libraries for Progressive Web Apps (PWAs)
- Google Maps API — A set of APIs for integrating Google Maps into your app
